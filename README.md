# ホル式戦術対抗戦マップ　-シッテムの箱-
**The FOR Map for pvp -Shittim no hako-**  
**FOR식 전술 대항전 맵 -싯딤의 상자-**

ブルーアーカイブの対抗戦（PvP）向け戦術シミュレーターツールです。  
A tactical PvP simulation tool for Blue Archive.  
블루 아카이브 대항전(PvP)을 위한 전술 시뮬레이터 툴입니다.

---

## 概要

マップ上に青点・赤点を配置し、以下の情報を視覚的に確認できます。

- 各ユニットの位置関係・部隊重心・最短距離
- 攻撃SP / バフSP / 攻撃ST / バフST のターゲット強調
- スキル範囲の描画（40種以上のキャラプリセット対応）
- キャラアイコンの割り当て・表示
- 配置のプリセット保存（12枠）・URL共有

---

## 対応環境

- **推奨：PCブラウザ（Chrome / Edge / Firefox）**
- モバイルでも一部動作しますが、PC表示を推奨します
- ブラウザのズームを90%程度にすると画面に収まりやすくなります

## 主な機能

### 基本操作
- 左クリックで点を配置、ドラッグで移動、右クリックで削除
- 青/赤のロッカースイッチで配置するチームを切り替え
- 召喚物を考慮するオプションで各チーム最大5点まで配置可能

### モード
| モード | 内容 |
|--------|------|
| 攻撃SP | SPアタッカーの攻撃ターゲットを強調 |
| バフSP | SPバッファーのバフターゲットを強調 |
| 攻撃ST | 基準点STの攻撃ターゲットを強調 |
| バフST | 基準点STのバフターゲットを強調 |
| 範囲確認 | 任意のスキル範囲を描画・確認 |

### プリセット・共有
- 配置を12スロットに名前付きで保存
- URLコピーで配置をそのまま共有（アイコン割り当て情報も含む）
- メモ欄（50字）をURLに埋め込んで状況を記録

### マップ
- **S10**・**S11** をワンクリックで切り替え
- 遮蔽物/ドラム缶レイヤーの表示/非表示切り替え

### 表示モード
| モード | 内容 |
|--------|------|
| ライト | 白背景の標準表示 |
| ダーク | ダークモード |
| アロナ | 背景画像付きの半透明UI |
